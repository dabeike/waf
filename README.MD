# 基于NodeJS的web应用防火墙（waf）

------

兼顾前端和服务器端的waf，能有效抵御SQL注入攻击以及XSS攻击。创新点在于前端防火墙的设计及实现，并且捕获可能存在的漏洞点，实时预警。

当前实现了：
> * 基于NodeJS的代理服务器
> * 前端防火墙实现（javascript） by friday
> * SQL注入过滤（NodeJS） by cyh
